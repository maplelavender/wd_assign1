My CSS Animations!

About the Animations

I've implemented very simple CSS animations in my website since I was going for a simple design and felt that any major animations would detract from that. The two animation styles I used are a colour changing type and a position changing changing type.

Colour Changing Animation

I added a colour changing animation to the navigation bar and did it differently between the "Desktop Style" and the "Mobile Style". In the desktop version, I added the animation to when you hover over the navigation bar, so instead of the background colour immediately changing, there's a gradual, fade-in of colour.

You can see the animation when you slowly hover over the Navigation bar.
On top of this, in the mobile version, I added a colour changing animation to the active page in the nav'bar.
Alongside this change I removed the italic styling since it didn't look as nice when the text in the nav'bar stacks when the window is very small.

Position Changing Animation

If you haven't noticed, there is an animation added to the footer so that the text shakes a little, I feel it adds a cute touch to the footer without the movement being too distracting.
Though, I did increase the movement in the mobile version, which I counterbalanced by making the animation duration longer.
